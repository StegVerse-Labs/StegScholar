# StegScholar Canon Overlay (StegDB)

This folder is synced from `stegverse-labs/stegdb` as an **overlay**.

## What this overlay controls
- Canonical doctrine and research principles
- Stable terminology/glossary stubs
- Submission standards templates
- Paper scaffolding templates (optional)

## What this overlay never touches
- `papers/**` content
- `references/**`
- `submissions/**`
- `ROADMAP.md`

## Release discipline
This overlay is intended to be pulled by tag/SHA at release intervals.
